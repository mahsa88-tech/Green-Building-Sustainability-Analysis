# 🌿 Green Building Sustainability Analysis

## Overview
Analysis of 40,000 buildings to identify the biggest 
drivers of carbon emissions and sustainability performance.
Built as a 5-page interactive Power BI dashboard.

##  Key Findings
-  Renewable HVAC reduces carbon by **53%** (7 vs 15 tons)
-  Eco insulation reduces emissions by **67%** (6 vs 18 tons)
-  LEED buildings save **5 tons CO₂/year** vs uncertified
-  Industrial sector = only **28% green** (biggest opportunity)
-  Grade A buildings emit **90% less** carbon than Grade E

## Dashboard Pages
1. Executive Overview — 40K buildings KPIs & green split
2. Carbon & Energy Performance — trends by year & HVAC
3. Green Features Impact — solar, insulation, energy grade
4. Certification Analysis — LEED vs BREEAM vs GreenMark
5. Recommendations — data-driven action plan

## Tools Used
- Power BI Desktop
- DAX measures
- Python (data preparation & enrichment)
- Dataset: 40,000 buildings
